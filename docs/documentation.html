In this file I will explain how to use my design system


typography

First to use headings, simply use the classes heading-1, heading-2 or heading-3. Please use heading-1 for titles, heading-2 for section titles within a page and heading-3 for titles to paragraphs.

Second is the body. use the classes body-1, body-2, body-3. body-1 is the largest, while body-3 is the smallest. Use these classes as you see fit.
